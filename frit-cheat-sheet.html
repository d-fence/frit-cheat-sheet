Frit cheat sheet

Initialize working directory

$ frit init

This will create a .frit directory in the current directory, alongside with a basic empty cofig file.

Add an evidence

$ frit add evidence_file_name

This command will add the file "evidence_file_name" into the .frit/config file. Frit will try to discover the file format and filesystems contained in evidence. Known containers formats:

  • Simple .dd
  • Encase E01
  • Aff
  • Directory
  • iso file

Important files

  • Everything goes in ".frit" directory
  • configuration file is in ".frit/config" file
  • Mounted filesystems are in ".frit/filesystems"
  • Mounted containers are in ".frit/containers"
  • ...

Mount

$ frit mount

This command will mount every filesystems from every evidences.

$ frit mount Evidence3

This command will mount every filesystems found on Evidence3 container.
